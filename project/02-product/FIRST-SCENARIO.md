# Первый продуктовый сценарий

**Статус:** пользовательский сценарий заблокирован до реального урока. Технический tracer bullet определён отдельно.

## Технический tracer bullet

Без привязки к предметному рынку:

1. преподаватель создаёт `Lesson Session`;
2. ученик открывает invitation link;
3. оба входят в одну video room;
4. оба видят общий объект — тестовое уравнение;
5. отображаются два курсора/presence;
6. изменение объекта синхронизируется между двумя браузерами;
7. состояние сохраняется;
8. после reload оба видят последнюю версию;
9. завершение session создаёт минимальный outcome.

Уравнение здесь является тестовым shared object, а не заявлением о выборе математики как первого рынка.
