# Текущее состояние продукта

**Статус:** не заполнено. Это главный блокер проекта.

Заполнять только с доказательствами.

| Возможность | Readiness | Что делает преподаватель | Что делает ученик | Что сохраняется | Ограничения | Доказательство |
|---|---|---|---|---|---|---|
| _пример строки_ | WORKING / PROTOTYPE / PLANNED / HYPOTHESIS |  |  |  |  | URL / экран / видео |

## Определения

- `WORKING` — реализовано и можно продемонстрировать.
- `PROTOTYPE` — кликабельная или визуальная модель, не рабочий продукт.
- `PLANNED` — принято в scope, но не реализовано.
- `HYPOTHESIS` — идея, которую ещё нужно проверить.

## Запрещено

Не добавлять возможность на сайт или в deck без строки в этой таблице и подтверждающего артефакта.
