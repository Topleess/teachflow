# Продажи и презентационные материалы

**Статус:** paused до утверждения нового продуктового сценария.

Сохранённые материалы первой версии лежат в `legacy-v1/`. Они являются исходными данными, а не одобренным клиентским комплектом.

Новая версия должна полностью совпадать с лендингом по:

- аудитории;
- сценарию;
- степени готовности;
- границам пилота;
- CTA.
