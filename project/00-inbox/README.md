# Inbox

Сюда складываются необработанные голосовые заметки, ссылки, документы, идеи и запросы.

Для каждого элемента указать:

- дата;
- источник;
- краткое содержание;
- предполагаемое направление;
- требуется ли решение.

После разбора элемент переносится в соответствующую папку или в `99-archive/`.
