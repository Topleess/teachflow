# TeachFlow · статус проекта

**Состояние:** product reset  
**Дата обновления:** 2026-07-19  
**Главный фокус:** определить продукт на одном реальном сценарии до следующей переработки сайта.

## Кратко

| Направление | Статус | Комментарий |
|---|---|---|
| Продукт | `BLOCKED` | Нет подтверждённого списка работающих возможностей и реального кейса урока |
| Исследование | `NOT STARTED` | Нужен первый преподаватель и один реальный урок |
| Сайт | `REJECTED / LIVE PREVIEW` | Технически работает, содержательно отклонён; не отправлять клиентам |
| Продажи | `PAUSED` | Материалы первой версии не использовать до новой продуктовой истории |
| Лиды | `PAUSED` | 7 публичных контактов проверены; исходящих сообщений не было |
| Инфраструктура | `WORKING` | HTTPS, контейнер и GitHub работают |

## Что уже существует

- опубликованный статический лендинг;
- GitHub-репозиторий;
- дизайн-направление, которое в целом принято;
- первая презентация, one-pager и сообщения, которые требуют пересборки;
- 7 проверенных публичных контактов;
- подробный план product reset.

## Главный блокер

Нет фактической основы для содержательного сайта:

1. что работает в продукте сейчас;
2. кто первый конкретный пользователь;
3. как выглядит один настоящий урок;
4. где именно TeachFlow меняет процесс;
5. что входит в первый пилот.

## Ближайшая контрольная точка

Заполнены:

- `02-product/CURRENT-STATE.md`;
- `01-strategy/INITIAL-SEGMENT.md`;
- `03-research/REAL-LESSON-01.md`.

После этого можно формулировать первый продуктовый сценарий.
