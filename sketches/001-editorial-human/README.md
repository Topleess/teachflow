## Variant: Editorial Human

### Design stance
Премиальная редакционная упаковка: сначала обещание и атмосфера, затем продуктовый сценарий.

### Key choices
- Layout: центрированный hero + широкий демонстрационный модуль + асимметричные карточки
- Typography: крупный нейтральный гротеск
- Color: тёплый белый, графит, серо-бежевый
- Interaction: форма пилотного партнёра

### Trade-offs
- Strong at: эмоциональность, доверие, презентационность
- Weak at: продукт объясняется чуть медленнее

### Best for
- Холодные знакомства, партнёрские презентации, небольшие школы
