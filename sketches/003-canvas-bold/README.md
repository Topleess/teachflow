## Variant: Canvas Bold

### Design stance
Урок показан как живой совместный холст, а не очередная LMS.

### Key choices
- Layout: типографический hero + свободная композиция учебных блоков
- Typography: гротеск с точечным editorial serif
- Color: кремовый, графит, охра, зелёный, коралл
- Interaction: форма пилотного партнёра

### Trade-offs
- Strong at: запоминаемость, продуктовый характер, демонстрация интерактивности
- Weak at: смелее и менее универсально для консервативных школ

### Best for
- Независимые преподаватели, авторские школы, ранние последователи
