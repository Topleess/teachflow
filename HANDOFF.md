# TeachFlow · handoff

## What is ready

A final responsive landing page combining the premium editorial composition of concept 1 with the collaborative lesson canvas of concept 3.

The page includes:
- clear product positioning;
- visual lesson scenario;
- preparation → live lesson → saved result → homework flow;
- audiences;
- pilot CTA and accessible modal with a working Telegram link;
- mobile transformation and reduced-motion support.

## Decisions intentionally left to the receiving team

1. **Optional lead form:** keep Telegram-only contact or later connect a CRM, email, Telegram bot or custom API.
2. **Real owner/legal details** if the project starts collecting personal data directly.
3. **Privacy-policy URL and consent wording** before adding a web form or analytics.
4. **Product truth:** confirm which demonstrated features exist in the prototype and adjust copy if needed.
5. **Analytics:** add only after choosing a provider and consent policy.
